Fractory Tutorial

Creating Shapes

Click & drag between two nodes to draw a line segment.
Click on a blank node to create a root node.
Click on a line segment to destroy it.

The root node marks the origin of a shape.

Creating Fractals

Shift-click a blank node (or normal-click a root node) to create a branch node.

At each branch node, the shape will be drawn again from its roots. This process is done recursively, creating fractals.

Clicking on a node multiple times will cycle it between normal, root, and branch.
Shift clicking will cycle in the other direction.

WARNING: Do not have too many root nodes and branch nodes active at the same time!
The Fractory will become very slow if (# of root nodes) x (# of branch nodes) exceeds 9 or so.

Transformation

Click & drag node arrows to rotate them.
Hold down shift to scale instead.
Hold down alt/option to enable smooth rotating/scaling.

Coloring

Press Space to randomize a fractal's color scheme & width.
Press shift+Space to revert to a previous color scheme (up to 25 times).
